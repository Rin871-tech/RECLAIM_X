# RECLAIM-X

**Smart Campus Resource Wastage Reporting System**

## Overview
RECLAIM-X allows students/staff to report campus resource wastage (Electricity, Water, Infrastructure). Admins view categorized reports, update status, and analyze resource-wise trends.

## Features
- Phone-based login (credential)
- Report submission (resource, location, description)
- Admin dashboard with resource-wise columns and status updates
- In-app notifications for resolved issues
- Analytics using 3 horizontal pie charts
- Firebase (Firestore) as a cloud-ready datastore with local CSV backup
